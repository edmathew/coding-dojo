# 4. Basic Types

## 4.1. Numeric

### 4.1.1. Integers

#### 4.1.1.1 Operators

| Operator | Operation      |
|:--------:|:---------------|
| +        | Addition       |
| -        | Subtraction    |
| *        | Multiplication |
| /        | Division       |
| %        | Modulus        |
| **       | Exponent       |

| Operator | Operation            |
|:--------:|:---------------------|
| =        | Assignment           |
| +=       | Increment itself by  |
| -=       | Decrement itself by  |
| *=       | Multiply itself by   |
| /=       | Divide itself by     |
| %=       | Modulus itself with  |
| **=      | Exponent itself by   |

> The increment and decrement operators ( ++ and -- ) are not available, neither in "pre" nor "post" forms.

| Operator | Operation               |
|:--------:|:------------------------|
| &        | Bitwise AND             |
| |        | Bitwise OR              |
| ^        | Bitwise XOR             |
| ~        | Bitwise Ones Complement |
| <<       | Bitwise Left Shift      |
| >>       | Bitwise Right Shift     |
